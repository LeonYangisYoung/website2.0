About This Portfolio

This website is part of my CCT260 Project 2 — a personal portfolio site created using HTML and CSS. I designed and coded all pages and produced all media assets myself (photos, screenshots, and graphics).

My goal is to present my work in a clean, simple, and accessible layout while following usability and accessibility guidelines such as WebAIM and WCAG 2.0.
